# 🪐 InfraUniverseRegistry

This project maps the **infrageometric landscape** of the [Wolfram Notable Universes](https://www.wolframphysics.org/universes/).

* We study hypergraph states under generational rewriting with [WolframModel](https://resources.wolframcloud.com/FunctionRepository/resources/WolframModel/).
* TBD: Causal infrageometry.
* TBD: Branchial infrageometry from multicomputation ([Hypergraph Rewriting Engine](https://github.com/WolframInstitute/HypergraphRewritingEngine), [Multicomputation](https://github.com/WolframInstitute/Multicomputation)).

## 🧪 Computational Experiments

* **Ball Volume Growth**: [Dimension and Curvature Landscape](https://www.wolframcloud.com/obj/hajek_pavel/InfraUniverseRegistry/VolumeGrowth), [Single Rule Ball Volume Growth](https://www.wolframcloud.com/obj/hajek_pavel/InfraUniverseRegistry/VolumeGrowthSingle).
