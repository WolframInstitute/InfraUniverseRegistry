# InfraUniverseRegistry

This project maps the **infrageometric landscape** of the [Wolfram Notable Universes](https://www.wolframphysics.org/universes/).

* We study hypergraph states under generational rewrites using [WolframModel](https://resources.wolframcloud.com/FunctionRepository/resources/WolframModel/).
* TBD: Multicomputation ([Hypergraph Rewriting Engine](https://github.com/WolframInstitute/HypergraphRewritingEngine), [Multicomputation](https://github.com/WolframInstitute/Multicomputation)).

## Computational Experiments

* [Volume Growth — Dimension and Curvature](https://www.wolframcloud.com/obj/hajek_pavel/InfraUniverseRegistry/VolumeGrowth) — all rules in the dimension–curvature landscape, plus a table of the 20 largest, most stable.
* [Volume Growth — Single Rule](https://www.wolframcloud.com/obj/hajek_pavel/InfraUniverseRegistry/VolumeGrowthSingle) — one rule's full geometric characterization.
